🧠 ATIVIDADE COMPLEMENTAR – C#
## 1. Conversor de Medidas
Escreva um programa em C# que peça ao usuário um valor em metros (double).
Converta esse valor para centímetros e milímetros e exiba os resultados no console.
________________________________________
## 2. Calculadora de Desconto
Crie um programa que receba o preço de um produto e um percentual de desconto.
Calcule o novo valor e exiba uma mensagem formatada usando Console.WriteLine com formatação (ex: F2 para casas decimais).
________________________________________
## 3. Analisador de Nome
Peça ao usuário para digitar seu nome completo.
Mostre:
•	Quantidade de caracteres (Length)
•	Nome em letras maiúsculas (ToUpper())
________________________________________
## 4. Cálculo de Idade em Dias
Solicite o nome e a idade (int) de uma pessoa.
Exiba:
Olá [nome], você já viveu aproximadamente [idade * 365] dias
________________________________________
## 5. Reajuste Salarial
Receba o salário de um funcionário e aplique um aumento de 15%.
Exiba o salário antigo e o novo salário.
________________________________________
## 6. Par ou Ímpar
Peça um número inteiro ao usuário.
Utilize if/else e o operador % para informar se o número é par ou ímpar.
________________________________________
## 7. Maior de Três (Sem funções prontas)
Peça três números inteiros ao usuário.
Sem usar Math.Max(), utilize if para descobrir:
•	O maior número
•	O menor número
________________________________________
## 8. Tabuada com Laço For
Peça um número ao usuário.
Exiba a tabuada de 1 a 10 usando for.
________________________________________
## 9. Validador de Senha com While
Crie um programa que solicite uma senha ao usuário.
Enquanto a senha for diferente de "Java2026":
•	Exiba: Senha incorreta
•	Peça novamente
Quando acertar:
•	Exiba: Acesso garantido
________________________________________
## 10. Estatísticas de Array
Peça ao usuário para inserir 5 números em um array.
Depois, mostre:
•	Maior valor
•	Menor valor
•	Média dos valores
________________________________________
## 11. Busca em Lista (List)
Crie uma List<string> com 5 produtos.
Peça ao usuário um nome de produto e informe se ele está na lista (Contains()).
________________________________________
## 12. Desafio Final – Sistema de Cadastro
Crie uma classe Aluno com:
•	Nome
•	Idade
•	Array de 3 notas
No Main:
•	Instancie um aluno
•	Peça os dados via Console.ReadLine()
•	Crie um método que calcule a média
Exiba:
•	Média final
•	"Aprovado" (média ≥ 7) ou "Reprovado"
